# BCIC2020T3-preprocessing
Data Preprocessing of 2020 International BCI Competition Track 3 

Step 1: Download 2020 International BCI Competition Track 3 from https://osf.io/pq7vb/

Step 2: Unzip the raw Track 3 MAT-data package under .\BCIC2020T3 preprocessing\data\0_raw_data\Track#3 Imagined speech classification

Step 3: Open "BCIC2020T3 preprocessing" folder as a project, and run step1_preprocess.py
